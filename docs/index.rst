Python Projects
==============================

Bienvenue dans la documentation technique du **Python Projects**. 
Ce projet regroupe plusieurs outils et divertissements développés en Python, accessibles via une interface centralisée en PyQt5.

.. include:: modules.rst
   :start-line: 2

Organisation du Projet
----------------------

Le hub est divisé en plusieurs modules indépendants que vous pouvez explorer ci-dessous :

* **Interface Principale** : Gestion du tableau de bord (``main.py``).
* **Module Jeux** : Regroupe l'ensemble des divertissements interactifs.
* **Module Calculatrice** : Outils de calcul mathématique.
* **Module Musique** : Lecteur et utilitaires audio.
